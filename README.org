
* =rubberstamp= fetches licenses for your project

=rubberstamp= fetches license texts from https://choosealicense.com and prints
them to =stdout=.

Here's how I use it:

#+begin_src 

$ # get a help message
$ ./rubberstamp.ros

USAGE: rubberstamp LICENSE
